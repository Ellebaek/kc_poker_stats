# Kings Club Stats
This repo holds 

- the R Markdown file `poker_pub.Rmd` to knit the Kings Club Poker Championship html page hosted on RPubs [rpubs.com/Ellebaek/kingsclub](https://rpubs.com/Ellebaek/kingsclub)
- the complete tournament "database" `tournament_results.csv`

# Poker Clock
An associated repo [github.com/Ellebaek/kk_poker_clock](https://github.com/Ellebaek/kk_poker_clock) contains source code for the simple poker clock used for Kings Club home poker tournaments. The Poker Clock is available at [ellebaek.se/clock](https://ellebaek.se/clock/).

# Note
Kings Club is a private poker club. All poker tournaments referenced in this repository are private, non-commercial arrangements between individuals.
No public events, services, or gambling operations are organized through this project.
The repository exists solely to document and analyze results for personal use.
