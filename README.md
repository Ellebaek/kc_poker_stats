# Kings Club Stats
This repo holds 

- the R Markdown file to knit the Kings Club Poker Championship html page hosted on RPubs [rpubs.com/Ellebaek/kingsclub](https://rpubs.com/Ellebaek/kingsclub)
- the complete tournament "database" (`tournament_results.csv`)

# Poker Clock
An associate repo [github.com/Ellebaek/kk_poker_clock](https://github.com/Ellebaek/kk_poker_clock) holds the source code for the simple poker clock used for Kings Clubs home poker tournaments.

# Note
Kings Club is a private poker club. All poker tournaments referenced in this repository are private, non-commercial arrangements between individuals.
No public events, services, or gambling operations are organized through this project.
The repository exists solely to document and analyze results for personal use.
